## Behaviors (Správania)

### Home Row Mods
Klávesy v prostrednom rade majú dvojitú funkciu:
- **Tap** (krátke stlačenie) = písmeno
- **Hold** (dlhé stlačenie 200ms+) = modifier

```
Ľavá ruka:          Pravá ruka:
A = Shift           J = Gui
S = Ctrl            K = Alt
D = Alt             L = Ctrl
F = Gui             ; = Shift
```

### Layer Tap (Thumb klávesy)

| Pos | Tap | Hold |
|-----|-----|------|
| 30 | Tab | SYM layer |
| 31 | Space | NAV layer |
| 32 | Escape | - |
| 33 | " | SYM layer |
| 34 | Backspace | NUM layer |
| 35 | Enter | FUN layer |

### Sticky Key (sk)
- Stlačíš modifier, pustíš, ďalší kláves bude s modifierom
- Auto-release po 750ms

### Caps Word
- Aktivuje CAPS pre jedno slovo
- Automaticky sa vypne pri medzere
- Pokračuje cez `-`, `_`, `Backspace`

## Modifier kombinácie

Pre kombináciu modifierov použite:
- `LS(x)` = Left Shift + x
- `LC(x)` = Left Ctrl + x
- `LA(x)` = Left Alt + x
- `LG(x)` = Left Gui + x
- `RS(x)`, `RC(x)`, `RA(x)`, `RG(x)` = pravé modifikátory

Príklady:
- `LC(C)` = Ctrl+C
- `LS(LC(ESC))` = Shift+Ctrl+Esc
- `LA(TAB)` = Alt+Tab
